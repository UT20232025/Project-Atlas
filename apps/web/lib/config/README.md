# Genwelth AI

Genwelth AI er en AI-drevet analyseplattform for kryptovaluta.

Målet er å gi tradere en rask oversikt over markedet ved hjelp av tekniske indikatorer, markedsdata og Atlas Engine.

---

# Tech Stack

- Next.js 16
- React
- TypeScript
- Tailwind CSS v4
- shadcn/ui
- Binance API
- CoinGecko API
- TradingView Widget

---

# Atlas Engine

Atlas Engine analyserer markedet ved hjelp av:

- Price Action
- RSI
- EMA20
- EMA50
- MACD (kommer)
- ATR (kommer)
- Volume
- Fear & Greed
- BTC Dominance
- Funding Rate (kommer)
- Open Interest (kommer)

Resultatet blir:

- Atlas Score
- LONG / WAIT / SHORT
- Confidence
- Reasons
